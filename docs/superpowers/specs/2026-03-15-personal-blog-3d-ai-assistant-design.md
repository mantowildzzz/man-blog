# 个人博客网站 + 3D AI 助手设计文档

**日期**：2026-03-15
**项目**：个人博客网站 with 3D AI 虚拟助手

---

## 1. 项目概述

- **项目类型**：个人博客网站
- **核心功能**：支持博客文章、视频、音频/播客内容发布，配备可交互的 3D AI 虚拟助手
- **目标用户**：访客、读者、内容消费者

---

## 2. 技术栈

| 层级 | 技术选择 |
|------|----------|
| 前端框架 | Nuxt 3 (Vue.js) |
| 3D 渲染 | Three.js + @pixiv/three-vrm |
| AI 对话 | Claude API / OpenAI API |
| 语音合成 | ElevenLabs TTS / Azure TTS |
| 样式 | Tailwind CSS |
| 部署 | Vercel / Netlify |

---

## 3. UI/UX 设计

### 3.1 整体风格

- **主题**：暗黑科技风
- **背景**：深色 (#0a0a0f, #121218)
- **点缀色**：霓虹蓝 (#00d4ff)、霓虹紫 (#8b5cf6)、霓虹粉 (#ff00aa)
- **字体**：无衬线字体，现代感

### 3.2 页面结构

#### 首页
- 顶部：导航栏（Logo + 菜单）
- 主区域：最新内容展示（轮播 + 网格）
- 底部：页脚

#### 内容分类页
- 博客文章列表
- 视频列表（带缩略图）
- 音频/播客列表（带封面）

#### 内容详情页
- 博客：Markdown 渲染，代码高亮
- 视频：嵌入式播放器
- 音频：带封面的播放器

### 3.3 响应式设计

- 桌面端：1920px+ 优化
- 平板端：768px-1919px
- 移动端：< 768px

## 4. AI 助手功能

### 4.1 3D 虚拟人物

- **模型格式**：VRM (.vrm)
- **形象来源**：用户提供照片，定制或使用现成模型
- **渲染**：Three.js + VRM 加载器
- **动作**：待机动画、行走动画（随机移动时触发）

### 4.2 随机走动行为

- **初始位置**：页面随机位置出现
- **移动逻辑**：
  - 每隔 5-15 秒随机移动到新位置
  - 移动时间 2-4 秒
  - 移动路径：贝塞尔曲线，模拟自然行走
  - 边界检测：不会移出视口
- **层级**：固定在内容层之上，UI 层之下

### 4.3 对话功能

- **触发方式**：点击虚拟人物展开对话窗口
- **对话窗口**：
  - 位置：跟随虚拟人物或固定在底部
  - 功能：文本输入、发送按钮
  - 展示：AI 回复内容
- **AI 模型**：Claude 或 OpenAI GPT-4
- **上下文**：可记忆对话历史

### 4.4 语音功能

- **TTS（文字转语音）**：
  - AI 回复时自动朗读
  - 可开关语音
  - 语音选择：多种音色可选
- **语音合成服务**：ElevenLabs 或 Azure TTS

## 6. 数据存储与成本控制

### 6.1 内容存储

| 内容类型 | 存储方案 |
|----------|----------|
| 博客文章 | Nuxt Content 模块（本地 Markdown 文件） |
| 视频 | YouTube/Vimeo 嵌入（推荐）或自建 CDN |
| 音频/播客 | 自建对象存储 + CDN，或第三方托管（SoundCloud） |

### 6.2 成本控制策略

- **AI 对话**：
  - 每日调用配额限制（如 1000 次/天）
  - 对话缓存：相同问题不重复调用 API
  - 建议使用 GPT-3.5 Turbo 降低费用
- **TTS**：
  - 仅在用户主动点击"朗读"时触发
  - 缓存已生成的语音文件
- **免费额度**：新用户赠送体验次数

### 6.3 性能优化

- **VRM 模型**：懒加载，进入页面 3 秒后开始加载
- **移动端降级**：移动端禁用 3D，改用静态图片
- **首屏优化**：先展示 UI，3D 模型后台加载
- **loading 状态**：模型加载时显示 skeleton

### 6.4 安全策略

- **API 密钥**：所有 AI/TTS API 调用通过 Nuxt Server Routes（服务端），前端不暴露密钥
- **输入过滤**：用户输入进行安全过滤，防止注入
- **速率限制**：服务端添加 API 速率限制

## 7. API 设计

### 7.1 内容 API

- `GET /api/posts` - 获取文章列表
- `GET /api/posts/:id` - 获取文章详情
- `GET /api/videos` - 获取视频列表
- `GET /api/audios` - 获取音频列表

### 7.2 AI 对话 API

- `POST /api/chat` - 发送消息，获取 AI 回复
- `POST /api/tts` - 文字转语音

---

## 8. 对话系统细节

### 8.1 对话窗口

- **位置**：固定在页面底部居中（优先级高于跟随虚拟人物）
- **展示**：点击虚拟人物展开，可最小化
- **历史存储**：sessionStorage（单会话），刷新页面清除

### 8.2 错误处理

- AI 服务不可用：显示友好提示"暂时无法回答，请稍后重试"
- TTS 加载失败：静默失败，不影响对话流程
- 网络断开：显示离线提示
